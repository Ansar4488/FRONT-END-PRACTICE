3️⃣ Control Flow in JavaScript

1️⃣ if / else Statements

2️⃣ else if

👉 Multiple conditions can be checked.

3️⃣ switch / case

👉 Alternative to multiple if/else.
👉 Compares strictly (===).

🔄 Loops

Loops are used to repeat code until a condition is met.

4️⃣ for Loop
👉 Runs a block of code a fixed number of times.

5️⃣ while Loop
👉 Runs while the condition is true.

6️⃣ do...while Loop
👉 Runs at least once, even if the condition is false.

7️⃣ for...of Loop
👉 Used to loop through arrays, strings, or iterable objects.

8️⃣ for...in Loop
👉 Used to loop through object properties (keys).
