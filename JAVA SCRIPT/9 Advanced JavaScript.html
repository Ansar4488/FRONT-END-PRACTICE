9️⃣ Advanced JavaScript Topics

1️⃣ ES6 Features

(a) let & const
let → block-scoped, re-assignable
const → block-scoped, cannot be re-assigned

(b) Arrow Functions
👉 Shorter syntax for functions.

(c) Template Literals

👉 Use backticks (`) for string interpolation & multi-line strings.

(d) Destructuring

👉 Extract values from arrays/objects easily.

2️⃣ Spread & Rest Operators

👉 Both use ... but behave differently.
Spread → expands values

Rest → collects values into an array

3️⃣ Default Parameters

👉 Set default values if no argument is passed.

4️⃣ Promises

👉 Handle async operations (like API calls).

5️⃣ Async/Await

👉 Cleaner way to work with promises.
