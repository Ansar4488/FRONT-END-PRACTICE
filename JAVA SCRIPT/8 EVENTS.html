8️⃣ Events in JavaScript
👉 An event is an action that happens in the browser (e.g., click, key press, form submit).
👉 JavaScript can listen for events and run code in response.

1️⃣ Event Types

| Event | Trigger | | ----------- | ------------------------------------- | | `click` | When the user clicks an element | | `mouseover` | When the mouse hovers over an element | | `keydown` | When a key is pressed | | `submit` | When a form is submitted | | `change` | When an input value changes |

2️⃣ Add Event Listener

👉 Best practice = use addEventListener instead of inline onclick.

3️⃣ Event Object

👉 The event object (e or event) contains info about the event.
