1️⃣ Function Declaration
👉 Standard way of creating a function.

2️⃣ Function Expression
👉 Assign a function to a variable.

3️⃣ Arrow Functions (()=>{})
👉 Shorter syntax introduced in ES6.
👉 Especially useful for small, one-line functions.

4️⃣ Parameters and Arguments
👉 Parameters = placeholders inside function.
👉 Arguments = actual values you pass when calling.

5️⃣ Return Statement
👉 Functions can return values.
