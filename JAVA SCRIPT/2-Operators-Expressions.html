Operators & Expressions

Arithmetic Operators

2️⃣ Assignment Operators

| Operator | Example | Equivalent to |
| -------- | -------- | ------------- |
| `=` | `x = 5` | assigns 5 |
| `+=` | `x += 3` | `x = x + 3` |
| `-=` | `x -= 3` | `x = x - 3` |
| `*=` | `x *= 3` | `x = x * 3` |
| `/=` | `x /= 3` | `x = x / 3` |
| `%=` | `x %= 3` | `x = x % 3` |

3️⃣ Comparison Operators

| Operator | Example | Result |
| -------- | ----------- | ----------------------------- | | `==` | `5 == "5"` | `true` (only checks value) |
| `===` | `5 === "5"` | `false` (checks value & type) |
| `!=` | `5 != "5"` | `false` |
| `!==` | `5 !== "5"` | `true` |
| `>` | `5 > 3` | `true` |
| `<` | `5 < 3` | `false` |
| `>=` | `5 >= 5` | `true` |
| `<=` | `5 <=5` | `true` |
