❓ Frequently Asked Questions

What is HTML?

HTML stands for HyperText Markup Language. It is used to structure web pages.

Example:

<!DOCTYPE html>
<html>
  <head><title>My Page</title></head>
  <body>
    <h1>Welcome</h1>
  </body>
</html>

How do I add a link in HTML?

Use the <a> tag with an href attribute.

Example:

<a href="https://www.example.com">Visit Example</a>

How do I style HTML with CSS?

You can use the <style> tag or link an external stylesheet.

Example:

<style>
  body {
    background-color: #f0f0f0;
    font-family: Arial;
  }
</style>
